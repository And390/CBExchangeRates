### REST-сервис получения котировок от ЦБ России.

Сделан на Spring Boot, для сборки используется Gradle со стандартным плагином application.
Чтобы собрать проект, нужно запустить `gradle build` или сразу `gradle run` для сборки и запуска. В каталоге `build/distributions` будут лежать собранные архивы приложения.

В задании не указано, какой ответ должен возвращаться в случае ошибки, поэтому я дополнил API для этого случая:
```json
{
    "error": "Internal server error"
}
```
